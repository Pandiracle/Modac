# Modac
```
Modac в данном случае это шаблон, разработанный мной с уже готовой вёрсткой сайта от Modac
```
## Описание шаблона
```
Данный шаблон, позволяет пользователю динамически изменять контент на странице сайта.
Контент сайта состоит из различных блоков которые добавляются из админ панели, а затем заполняются контентом.
В шаблоне соблюдена валидация данных для более удобного добавления.
```
## Технологии
```
Для данного шаблона использовался только один плагин Advanced Custom Fields.
Данный плагин позволил мне создавать блоки для сайта со своими уникальными полями,
которые в последствии и послужили для создание динамического контента для сайта.
```
## Процесс добавления блоков
```
Чтобы добавить блок на сайт необходимо зайти на нужную страницу. 
В данном шаблоне страница одна так как это landing.
При помощи Gutenberg добавляешь блок на сайт в списке ищешь нужный тебе блок
(список всех блоков будет ниже в разделе готовые блоки).
После добавления если блок требует заполнить поля,
то необходимо будет из заполнить и обновить или опубликовать страницу.
```
## Готовые блоки
```
- about section starts.
Данная секция отвечает контент About Me до секции What I Offer
- hero section starts.
Данная секция отвечает за шапку и слайдер с контетом до секции about
- tabs section starts.
Данная секция отвечает контент What I Offer до секции Testimonial
- testimonial section starts.
Данная секция отвечает контент Testimonial до секции Some Inspiration
- portfolio section starts.
Данная секция отвечает контент Some Inspiration до секции Plans & Pricing
- pricing section starts.
Данная секция отвечает контент Plans & Pricing до секции Say Hello
- ctac section starts.
Отвечает за закрытие нижней части
- contact section starts.
Данная секция отвечает обратную форму
```
## Примичание
```
Данный шаблон не идеален и что-то придётся редактировать в коде самих блоков
```
